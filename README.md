# YOLO + PostgreSQL + FastAPI Integration

A scalable object detection API using YOLO and FastAPI, with PostgreSQL for structured data storage.

## Features
- RESTful API with FastAPI
- YOLO detection
- PostgreSQL backend
- Clean and modular design

## Tech Stack
- YOLO (Ultralytics)
- FastAPI
- PostgreSQL
- Python

## Use Case
Great for structured CV pipelines and enterprise-grade deployments.

## Installation & Launch

1. Clone the repo:
   ```bash
   git clone https://github.com/Mazen-Ahmed12/integerating_Yolo_with_Postgresql_and_fastAPI.git
   cd integerating_Yolo_with_Postgresql_and_fastAPI

2. Install dependencies:
   ```bash
   pip install -r requirements.txt

3. Launch FastAPI:
   ```bash
   uvicorn app:app --reload

4. Access API docs:
   - http://127.0.0.1:8000/docs
